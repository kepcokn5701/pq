기술요약

시스템 개요

본 시스템은 한국전력공사 경남본부 배전건설부 감리용역 적격심사(PQ) 업무를 지원하기 위한 자동 교차검증 도구입니다. 업체가 제출한 PDF(사업수행능력 평가서)와 Excel(심사표)을 분석하여 점수를 독립적으로 재계산하고, 기재값과의 불일치를 자동으로 감지합니다.

3방향 교차검증

검증 소스설명
기준 계산값심사기준표를 적용하여 시스템이 독립적으로 산출한 점수
PDF 추출값업체가 제출한 PDF에서 직접 추출한 기재 점수
Excel 제출값업체가 제출한 심사 득점표(Excel)의 점수

평가 항목별 점수 체계

항목배점세부 내용
1. 참여감리원50점책임/보조/비상주 감리원 등급, 전기분야 경력, 참여분야 경력
2. 유사용역 수행실적10점사정금액 대비 유사용역 수행 비율
3. 기술개발 및 투자실적10점개발실적(특허/실용신안/신기술), 투자실적, 교육실적
4. 업무중첩도10점타 용역과의 중복 여부 (상주/비상주)
5. 교체빈도5점감리업체/참여감리원 교체 빈도
6. 가점,감점+1~-5자격보유 가점, 부실벌점 감점

경력 점수 산정 방식

  • 공사비 구간 + 등급 기반: 책임감리원 참여분야 (총 예정전기공사비와 감리원 등급에 따라 기준년수 결정)
  • 공사비 구간만: 전기분야 경력 (등급 무관, 공사비 구간으로 기준년수 결정)
  • 고정 기준: 보조감리원 참여분야 (5년 고정 기준, 공사비/등급 무관)

검증 결과 표시

  • ✓ 일치 : PDF 추출값과 Excel 제출값이 허용 오차 이내로 일치
  • ⚠ 주의 : 경미한 차이 또는 확인 필요 항목
  • ✗ 불일치 : 기준 계산값 대비 기재값이 크게 상이하여 확인 필요

사용방법

1단계: 파일 업로드

  1. 폴더 선택 (권장): 업체별 PDF/Excel이 들어 있는 상위 폴더를 선택하면 자동으로 파일을 분류합니다.
  2. 개별 파일 선택: PDF와 Excel 파일을 각각 직접 선택할 수도 있습니다. Ctrl+클릭으로 여러 파일을 선택할 수 있습니다.

2단계: 옵션 설정

  • 입찰공고일: 해당 공고의 입찰공고일을 설정합니다. 경력년수 산정의 기준일이 됩니다.
  • 총 예정전기공사비: 공사비 구간을 선택합니다. 감리원 경력 점수 산정 기준년수에 영향을 줍니다.

3단계: 분석 실행

  1. [분석 시작] 버튼을 클릭합니다.
  2. 업체별로 PDF 파싱, 점수 추출, 기준 계산, Excel 비교가 순차적으로 진행됩니다.
  3. 진행 상태가 화면에 표시됩니다.

4단계: 결과 확인

  • 요약 카드: 전체 검증 결과 통계 (일치/불일치/경고 건수)
  • 점수 현황: 항목별 점수를 막대 그래프로 표시
  • 교차검증 상세비교표: 8열 테이블로 항목별 기준계산값, PDF추출값, Excel제출값을 비교

결과 해석

컬럼배경색설명
기준 계산값노란색 배경심사기준표에 따라 시스템이 독립 산출한 점수. 근거 정보가 함께 표시됩니다.
PDF 추출값흰색업체 제출 PDF에서 자동 추출한 기재 점수
Excel 제출값흰색업체 제출 Excel 득점표의 점수

복수 업체 분석

여러 업체의 PDF/Excel을 동시에 업로드하면 업체별로 결과가 생성됩니다. 상세비교표 상단의 드롭다운에서 업체를 전환하여 각 업체의 결과를 확인할 수 있습니다.

유의사항

  • PDF 파일은 텍스트 추출이 가능한 파일이어야 합니다 (스캔본은 일부 항목 추출 불가).
  • Excel 파일은 한전 PQ 심사서류 득점표 양식이어야 합니다.
  • 기준 계산값과 기재값이 다르다고 반드시 오류는 아닙니다. 반드시 원본 서류를 확인하세요.
